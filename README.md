# Bookmark Duplicate Cleaner

[日本語](#日本語) | [English](#english)

---

## English

A Chrome extension to safely remove duplicate bookmarks.

### Features
- Fast duplicate bookmark detection
- Safe deletion with preview
- Local data processing (no server communication)
- Privacy-focused design

### Installation
1. Download from Chrome Web Store (coming soon)
2. Or install manually:
   - Download this repository
   - Open Chrome Extensions (chrome://extensions/)
   - Enable "Developer mode"
   - Click "Load unpacked" and select the folder

### Usage
1. Click the extension icon in Chrome toolbar
2. Click "重複スキャン開始" (Start Scan)
3. Review detected duplicates
4. Click "削除実行" (Delete) to remove duplicates

### Current Language Support
- **UI Language**: Japanese only (English version coming soon)
- **Documentation**: English + Japanese

### Privacy
- All processing happens locally on your device
- No data sent to external servers
- See [PRIVACY-POLICY.md](PRIVACY-POLICY.md) for details

### Development
- Built with Chrome Extensions Manifest V3
- Vanilla JavaScript (no frameworks)
- Professional logging and error handling

## 日本語

重複ブックマークを安全に削除するChrome拡張機能です。

### 機能
- 高速な重複ブックマーク検出
- プレビュー機能付きの安全な削除
- ローカル処理（サーバー通信なし）
- プライバシー重視の設計

### インストール
1. Chrome ウェブストアからダウンロード（準備中）
2. または手動インストール：
   - このリポジトリをダウンロード
   - Chrome拡張機能ページ (chrome://extensions/) を開く
   - 「デベロッパーモード」を有効化
   - 「パッケージ化されていない拡張機能を読み込む」でフォルダを選択

### 使い方
1. Chromeツールバーの拡張機能アイコンをクリック
2. 「重複スキャン開始」をクリック
3. 検出された重複を確認
4. 「削除実行」で重複を削除

### 現在の言語対応
- **UI言語**: 日本語のみ（英語版は今後対応予定）
- **ドキュメント**: 英語 + 日本語

### プライバシー
- すべての処理はお使いのデバイス上で実行
- 外部サーバーへのデータ送信なし
- 詳細は [PRIVACY-POLICY.md](PRIVACY-POLICY.md) をご覧ください

### 開発情報
- Chrome Extensions Manifest V3 使用
